# 🧠 Next Word Prediction using LSTM

This project demonstrates a Next Word Prediction system built using Long Short-Term Memory (LSTM) networks in TensorFlow and Keras. The model is trained on textual data to learn sequential patterns and relationships between words, enabling it to predict the most probable next word for a given input phrase.

The workflow includes text preprocessing, tokenization, sequence generation, padding of variable-length inputs, and one-hot encoding of target words. These processed sequences are then used to train an LSTM-based neural network capable of capturing contextual information and language dependencies.

This project provides a practical introduction to Natural Language Processing (NLP), sequence modeling, and deep learning techniques used in applications such as predictive text keyboards, chatbots, and text generation systems.

## ✨ Features
- Text preprocessing and tokenization
- Sequence generation and padding
- LSTM-based language modeling
- Next word prediction from input sequences
- End-to-end NLP workflow implementation

## 🛠️ Tech Stack
- Python
- TensorFlow
- Keras
- NumPy
- Jupyter Notebook

## 📌 Workflow
Text Corpus → Tokenization → Sequence Generation → Padding → LSTM Model Training → Next Word Prediction
